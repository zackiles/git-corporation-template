# Changelog Entry Template

Use this format when documenting corporate actions in a changelog or summary.

---

## [DATE] – [Short Description]

**PR:** #[NUMBER]

**Type:** [Director Change / Share Issuance / Policy Update / Meeting / Filing / Other]

**Resolution(s):** [RYYYY-NN, RYYYY-NN]

### Summary

[1-2 sentence description of the action]

### Affected Files

- `path/to/file1.md`
- `path/to/file2.csv`

### Register Updates

| Register | Change |
|----------|--------|
| `directors-register.csv` | Added [NAME] |
| `shareholders-register.csv` | Issued X shares to [NAME] |

### Related Filings

- [ ] Corporations Canada notice filed
- [ ] Provincial notice filed
- [ ] N/A

### Signed Documents

- [Document name](../09-binary-artifacts/path/to/document.pdf)

---

## Example Entry

## 2025-03-15 – Appointment of CFO

**PR:** #42

**Type:** Officer Appointment

**Resolution(s):** R2025-03

### Summary

Jane Smith appointed as Chief Financial Officer effective March 15, 2025.

### Affected Files

- `04-meetings-and-resolutions/board/2025/2025-03-15-q1-meeting/resolutions/R2025-03-appoint-cfo.md`

### Register Updates

| Register | Change |
|----------|--------|
| `officers-register.csv` | Added Jane Smith as CFO |

### Related Filings

- [ ] N/A (officer changes don't require filing)

### Signed Documents

- [R2025-03-appoint-cfo.pdf](../09-binary-artifacts/04-meetings-and-resolutions/board/2025/R2025-03-appoint-cfo.pdf)

# Meta Templates

> **Navigation**: [← Meta](../README.md) · [← Root](../../README.md)

Templates for repository processes (PR descriptions, changelogs), not corporate documents.

## Contents

- `_pull-request-description-template.md` - Template for PR descriptions
- `_changelog-entry-template.md` - Template for changelog entries

## Usage

These templates are for Git workflow processes, not corporate governance documents.
Corporate document templates are located in their respective folders (e.g., `04-meetings-and-resolutions/`).

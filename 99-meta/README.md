# Meta

This folder contains meta-process descriptions for the corporation minute book, not legal documents.

## Contents

- `governance-process.md` - PR-based governance workflow
- `CONTRIBUTING.md` - Technical guidelines for contributors
- `templates/` - Templates for changelog entries and PR descriptions

## Purpose

This folder documents:
- How to use the minute-book-as-code structure
- PR review requirements
- Branch naming conventions
- Technical contribution guidelines

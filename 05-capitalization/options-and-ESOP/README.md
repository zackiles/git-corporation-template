# Options and ESOP

This folder contains stock option plan documentation and individual grant records.

## Contents

- `esop-plan-template.md` - Template for the stock option plan
- `option-grant-template.md` - Template for individual option grants
- `option-grants-register.csv` - Register of all option grants

## ESOP Workflow

1. Adopt stock option plan (Board and Shareholder approval)
2. Authorize option pool in cap table
3. Issue individual grants using the template
4. Track grants in `option-grants-register.csv`
5. Store signed documents in `09-binary-artifacts/05-capitalization/options-and-ESOP/`

## Key Dates

Track these for each grant:
- Grant date
- Vesting start date
- Vesting schedule milestones
- Expiration date
- Exercise events

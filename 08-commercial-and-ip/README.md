# Commercial and IP

This folder contains key commercial contracts, intellectual property documentation, and HR-related IP agreements.

## Contents

- **key-contracts/** - Major commercial agreements (MSAs, NDAs)
- **ip/** - IP assignments and registrations
- **hr/** - Employee IP and confidentiality agreements

## What Belongs Here

Only "key" contracts should be stored here:
- Major customer Master Service Agreements
- Long-term vendor contracts
- Strategic partnership agreements
- IP assignments
- Material licensing agreements

Routine contracts and purchase orders do not belong here.

## Signed Documents

Executed versions of all agreements are stored in:
```
09-binary-artifacts/08-commercial-and-ip/...
```

Markdown files in this directory provide summaries and link to the signed PDFs.

- MSA skeleton
- NDA skeleton
- IP assignment skeleton

# HR (IP and Confidentiality)

Employee IP and confidentiality agreements.

## Purpose

This folder contains templates and records for:
- Employee IP assignment agreements
- Confidentiality agreements
- Non-compete/non-solicitation agreements (where permitted)

## Key Documents

Every employee should sign:
1. IP Assignment Agreement
2. Confidentiality Agreement
3. Acknowledgment of policies

## Templates

- `employee-ip-and-confidentiality-template.md` - Combined IP and confidentiality agreement

## Signed Agreements

Signed employee agreements stored in: `09-binary-artifacts/08-commercial-and-ip/hr/`

Organize by employee name or ID.

## Column Definitions

### Owned Properties

- **Property ID**: Internal reference number
- **Address**: Civic address
- **Legal Description**: Legal land description
- **Acquisition Date**: Date purchased
- **Purchase Price**: Amount paid
- **Current Use**: How property is used
- **Encumbrances**: Mortgages, liens, easements

### Leased Properties

- **Property ID**: Internal reference number
- **Address**: Civic address
- **Landlord**: Lessor name
- **Lease Start/End**: Term of lease
- **Monthly Rent**: Rent amount
- **Purpose**: How property is used

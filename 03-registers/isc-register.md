# ISC Register

Register of interests in land filed with the Information Services Corporation (ISC) or equivalent provincial land registry.

> Keep this register in sync with legal requirements and filings.

## Current Registrations

| Title/Parcel ID | Property Description | Interest Type | Registration Date | Registration Number | Status |
|-----------------|---------------------|---------------|-------------------|---------------------|--------|
| | | | | | |

## Column Definitions

- **Title/Parcel ID**: Land titles office parcel identifier
- **Property Description**: Legal description of the property
- **Interest Type**: Fee simple, leasehold, easement, mortgage, etc.
- **Registration Date**: Date registered with land registry
- **Registration Number**: Registration/instrument number
- **Status**: Active, Discharged, Transferred

# Regulatory Filings

> **Navigation**: [← Back to Root](../README.md) · [Workflows](../WORKFLOWS.md) · [Secretary Guide](../SECRETARY.md)

This folder contains documentation of regulatory filings with federal, provincial, and tax authorities.

## Contents

- **corporations-canada/** - Filings with Corporations Canada
- **ontario/** - Ontario provincial filings
- **cra/** - Canada Revenue Agency filings
- **licenses-and-permits/** - Business licenses and permits

## Signed Documents

Confirmation PDFs and scanned filings are stored in:
```
09-binary-artifacts/06-regulatory-filings/...
```

## Linking to Governance Documents

Each filing should be linked to any related governance documents:
- Director/officer changes → `03-registers/` and `04-meetings-and-resolutions/`
- Registered office changes → resolutions in `04-meetings-and-resolutions/`
- Annual returns → financial statements in `07-finance-and-tax/`

## Common Filing Deadlines

| Filing | Deadline | Authority |
|--------|----------|-----------|
| Annual Return | Within 60 days of anniversary | Corporations Canada |
| Ontario Annual Return | Within 60 days of anniversary | ServiceOntario |
| T2 Corporate Tax Return | 6 months after fiscal year end | CRA |
| HST Returns | Varies by reporting period | CRA |

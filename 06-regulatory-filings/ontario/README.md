# Ontario Filings

Filings with the Ontario provincial government.

## Contents

- **extra-provincial/** - Extra-provincial licence and registrations
- **annual-returns/** - Ontario annual information returns

## Extra-Provincial Registration

Federal corporations doing business in Ontario must register as an extra-provincial corporation.

## Common Filings

| Filing | When Required |
|--------|---------------|
| Initial Return (Form 2) | Within 60 days of starting Ontario operations |
| Annual Return | Annually |
| Notice of Change | When corporate information changes |

# Finance and Tax

This folder contains high-level finance and tax documentation, including board resolutions for financial matters.

## Contents

- **financial-statements/** - Board resolutions approving financial statements
- **tax-returns/** - Tax return filing records
- **banking/** - Bank account mandates and resolutions

## Scope

This folder is for:
- Board resolutions approving financial statements
- Tax return filing records and resolutions
- Bank account opening/closing resolutions
- High-level financial governance

This folder is NOT for:
- Day-to-day bookkeeping
- General ledger entries
- Invoices and receipts (see `09-binary-artifacts/receipts/`)

## Signed Documents

Actual financial statements and tax returns (PDFs) are stored in:
```
09-binary-artifacts/07-finance-and-tax/...
```

## Key Responsibilities

| Item | Frequency | Responsible |
|------|-----------|-------------|
| Financial statement approval | Annual | Board |
| T2 return filing | Annual | CFO/Accountant |
| HST returns | Per period | CFO/Accountant |
| Bank mandate updates | As needed | Board |

# Formation Documents

> **Navigation**: [← Back to Root](../README.md) · [Workflows](../WORKFLOWS.md) · [Secretary Guide](../SECRETARY.md)

This folder contains documents relating to incorporation and initial organization of the corporation.

## Contents

- **federal/** - Federal incorporation documents (articles, certificates)
- **ontario-extra-provincial/** - Ontario extra-provincial registration documents
- **organizational-resolutions/** - Initial board and shareholder resolutions
- **bylaws/** - Corporate bylaws

## What Belongs Here

- Articles of Incorporation
- Certificate of Incorporation
- Initial notices and filings
- Initial board resolutions (acknowledging first directors)
- Organizational (first) resolutions
- Founder share issuance resolutions
- Corporate bylaws and amendments

## Signed Documents

Signed versions (PDFs) of all formation documents are stored in:
```
09-binary-artifacts/01-formation/...
```

Each Markdown document should link to its corresponding signed PDF using relative paths.

## Key Documents Needed

Per Canadian corporate law, these formation documents are mandatory:

1. **Bylaws** - Defines how the corporation operates
2. **Initial Directors Resolution** - Acknowledges first directors
3. **Organizational Resolution** - The "birth certificate" adopting bylaws, appointing officers, authorizing share issuance
4. **Founder Share Issuance Resolution** - Specifies initial share allocation

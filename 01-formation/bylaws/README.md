This folder contains the corporation's bylaws and any amendments.

## Current Bylaws

- `_001-general-by-law-template.md` - Template for the general by-law

## Amendments

Amendments to bylaws should be:
1. Drafted as a new Markdown file
2. Approved via PR
3. Signed PDF deposited in `09-binary-artifacts/01-formation/bylaws/`
4. Linked from the Markdown file

## Naming Convention

```
NNN-description.md
```

Where `NNN` is a zero-padded sequence number.
